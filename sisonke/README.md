# Sisonke Task Manager API

Sisonke is a simple task management REST API built using Django and Django REST Framework.  
It allows users to register, login and manage their personal tasks.

## Features
- User Authentication
- Create Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks Completed

## Tech Stack
- Python
- Django
- Django REST Framework
- SQLite (Development)

## Setup

```bash
git clone https://github.com/yourusername/sisonke-task-manager-api.git
cd sisonke-task-manager-api
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
